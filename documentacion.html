Lista de comandos de TechBot

Aqui encontrarás todos los comandos disponibles, una descripcion para cada comando y un ejemplo de su uso


Información de uso


Prefijo: "t!", configurable.

Los corchetes [ ] indican que la opcion de dicho comando es opcional.
Los signos <> indican que la opcion de dicho comando es requerida.

Comandos de Configuración (7)

/configban - Canal para los baneos


Con este comando podrás configurar el canal para los logs de los baneos, solo disponible para administradores.
Uso:
/configban <canal>

/configbienvenidas - Canal y estado para las bienvenidas


Este comando sirve para configurar el modulo de bienvenidas, permitiendo activarlo o desactivarlo y eligir un canal para las bienvenidas, solo disponible para administradores.
Uso:
/configbienvenidas <activar> <canal>

/configdespedidas - Canal y estado para las despedidas


Este comando sirve para configurar el modulo de despedidas, permitiendo activarlo o desactivarlo y eligir un canal para las despedidas, solo disponible para administradores.
Uso:
/configdespedidas <activar> <canal>

/configlogs - Canal para los logs


Este comando sirve para configurar el canal en donde se enviarán todos los logs generales, solo disponible para administradores.
Uso:
/configlogs <canal>

/setprefix - Establece el prefijo


Con este comando podrás configurar el prefijo del bot, solo disponible para administradores.
Uso:
/setprefix <prefix>

/configroleadd - Establece el rol para usuarios


Este comando establece el rol a dar para los nuevos usuarios, tambien con la opcion de activarlo o desactivarlo, solo disponible para administradores.
Uso:
/configroleadd <rol> <status>

/toggleallevents - Activa o desactiva los eventos


Este comando activa o desactiva los eventos generales para todo el servidor, solo disponible para administradores.
Uso:
/toggleallevents <activar>

Comandos de Moderación (5)

/ban - Banea a un usuario


Este comando sirve para banear a un usuario del servidor, es para baneo permantente, solo disponible para administradores.
Uso:
/ban <usuario> <razon>

/clear - Elimina mensajes


Con este comando podrás eliminar una cantidad de mensajes en un canal o de un usuario, solo disponible para administradores.
Uso:
/clear <mensajes> [usuario]

/kick - Expulsa a un usuario


Con este comando se puede expulsar a un suaurio del servidor, solo disponible para administradores.
Uso:
/kick <usuario> [razon]

/mute - Silencia a un usuario


Aisla temporalmente a un usuario, solo disponible para administradores.
Uso:
/mute <usuario> <tiempo> [razon]

/unban - Desbanea a un usuario


Con este comando podrás desbanear usuarios baneados del servidor, solo disponible para administradores.
Uso:
/unban <userid>

Comandos de Diversión

t!8ball - Pregunta a la bola magica

Este comando es para hacerle una pregunta de si o no al bot.

t!azar - Elige una opcion al azar

Introduce un minimo de 2 opciones y el bot elegirá por ti.
